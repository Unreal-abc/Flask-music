# Flask-music
基于Flask框架的音乐网站
##### Python库安装
`
pip install flask`
`pip install validators
`
<br />
数据库系统采用SQLite。
### 项目特点
- ①不引入第三方组件库，完全自己实现。
- ②登录或注册，没输入或输入错误的账号密码会有提示。拦截SQL代码，密码采用sha256加密，数据库也不会存储明文密码。
- ③主界面采用Ajax动态刷新技术。
